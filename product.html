About Futility

Futility is a sorting system designed using 3 separate modules, which work together to sort resistors automatically.

Conveyor belt system

The conveyor belt system transports the resistors from the load point, towards the sorting modules. The fitted stepper motor is used for fine position control, allowing resistors to be aligned under the camera for identification.

The resistors, once scanned by the computer vision system, are carried to the sorting bins, where they are dropped after the correct bin is aligned with the conveyor belt based on the angle of the plate.

The conveyor belt is built using silicon for optimum grip. The frame and rollers are built from aluminium and use bearings, allowing for smooth and silent operation in any lab.

Computer vision system

Futility uses OpenCV-powered computer vision to detect the presence of a resistor, and to identify its value using the stripe colour code. This allows for fast scanning and does not require alignment or additional moving parts, allowing for uninterrupted, silent operation.

No jams guaranteed!*

"Wedge" storage system

An elegant sorting system has been designed using a rotating plate system with "wedge"-shaped containers. The containers can be removed to allow for easy retrieval of the sorted resistors, for instance to put back into organised storage.
